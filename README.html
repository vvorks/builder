HOW TO BUILD AND DEBUG

build

gradlew bootJar

run

java -jar build/vvorksbuilder-[version].jar

and access http://localhost:8080/gwt.html

debug

loadging

on linux

gradlew bootRun &
gradlew gwtCodeServer &

on windows

start gradlew bootRun
start gradlew gwtCodeServer

on eclipse

launch bootRun and gwtCodeServer tasks using 'Gradle task'

run

access http://localhost:8080/gwtDebug.html
